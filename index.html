A Beginner's Guide to GitHub

Learn about accounts, Git commands, and web hosting!

Steps to Create a GitHub Account

  1. Visit the official site: GitHub.com.
  2. Click the Sign up button at the top right.
  3. Enter your email address and create a strong password.
  4. Choose a unique username and verify your account via email.

Standard Git Commands

Command Description
git init Initializes a new local repository.
git add . Stages all modified files for a commit.
git commit -m "msg" Records changes to the repository with a message.
git push origin main Uploads your local commits to the remote GitHub repository.
git pull Fetches and merges the latest changes from the remote server.

Hosting a Repository and Blog on GitHub Pages

GitHub Pages allows you to host static websites for free directly from a r epository.

  • Create a repository named username.github.io .
  • Upload your web files. Make sure your homepage is named index.ht ml.
  • Navigate to your repository Settings, then click on Pages.
